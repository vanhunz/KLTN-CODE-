SILICON CURATOR - AUTHENTICATION PRESENTATION GUIDE
==================================================

1) MUC TIEU HE THONG AUTH
-------------------------
He thong authentication cua project nay gom 3 chuc nang chinh:
- Dang nhap (login)
- Dang ky (register) co OTP qua email
- Quen mat khau (forgot password) co OTP qua email

He thong dung:
- Node.js + Express o backend
- Prisma ORM de lam viec voi MySQL
- JWT de xac thuc dang nhap
- bcryptjs de ma hoa mat khau
- Nodemailer de gui OTP bang Gmail SMTP
- React + React Router o frontend

2) LUONG DANG NHAP
------------------
Dang nhap la quy trinh:
- Nguoi dung nhap email va password
- Backend kiem tra email co ton tai khong
- Backend so sanh mat khau da bam voi bcrypt
- Neu dung, backend tao JWT token
- Frontend luu token va thong tin user vao AuthContext
- Sau do chuyen huong den trang phu hop (Home, Profile, Dashboard)

API chinh:
- POST /api/auth/login
- GET /api/auth/me

3) LUONG DANG KY CO OTP
-----------------------
Dang ky duoc tach thanh 2 buoc:

Buoc 1:
- Nguoi dung nhap email
- Frontend goi API de gui OTP
- Backend tao OTP 6 chu so
- OTP duoc luu vao bang Email_Verifications
- OTP duoc gui vao email bang Nodemailer

Buoc 2:
- Nguoi dung nhap:
  + OTP
  + password
  + fullName
  + phone
  + address
- Backend kiem tra OTP dung va chua het han
- Neu hop le, backend bam password bang bcrypt
- Tao user moi trong bang Users
- Xoa OTP da su dung

API chinh:
- POST /api/otp/send-otp
- POST /api/otp/register

4) LUONG QUEN MAT KHAU
----------------------
Quen mat khau cung gom 2 buoc:

Buoc 1:
- Nguoi dung nhap email
- Backend kiem tra email co ton tai trong Users
- Tao OTP reset 6 chu so
- Luu OTP vao Email_Verifications
- Gui email noi dung reset password

Buoc 2:
- Nguoi dung nhap:
  + email
  + otp
  + newPassword
- Backend kiem tra OTP dung va chua het han
- Neu hop le, backend bam mat khau moi
- Update password trong Users
- Xoa OTP cu sau khi reset thanh cong

API chinh:
- POST /api/otp/forgot-password
- POST /api/otp/reset-password

6) DATABASE CHINH
-----------------
Cac bang lien quan:
- Users: luu thong tin tai khoan
- Email_Verifications: luu OTP va han su dung
- Roles: phan quyen user
- Cart, Orders, Reviews...: cac bang khac cua he thong

Bang Email_Verifications co cac cot:
- id
- email
- otp
- expiredAt
- createdAt

Bang Users co cac cot quan trong:
- id
- email
- passwordHash
- fullName
- phone
- address
- avatarUrl
- status
- roleId
- createdAt
- updatedAt

10) LUA CHON THIET KE LUONG DANG KY
-----------------------------------
Project nay dung mo hinh 2 buoc de dam bao:
- Email hop le
- Nguoi dung thuc su so huu email
- Giam spam tao tai khoan ao
- Co the dung cho reset mat khau sau nay

Ly do tach thanh 2 buoc:
- Buoc 1: gui va xac thuc email
- Buoc 2: nhap thong tin ca nhan + mat khau

11) LUU Y KHI TRINH BAY
----------------------
Khi giai thich voi giang vien, co the noi ngan gon:
- Dang nhap dung JWT de duy tri phien
- Dang ky phai co OTP de xac minh email
- Quen mat khau cung dung OTP de dam bao an toan
- Mat khau luon duoc bam bang bcrypt, khong luu plain text
- OTP co han 5 phut va duoc xoa sau khi su dung

12) LUONG CHAY TONG QUAT
------------------------
1. Nguoi dung mo frontend
2. Chon Login / Register / Forgot Password
3. Frontend goi API backend
4. Backend validate du lieu
5. Backend truy cap MySQL qua Prisma
6. Neu hop le thi tra JSON success
7. Frontend hien thong bao va chuyen trang

13) THONG TIN CHAY DU AN
